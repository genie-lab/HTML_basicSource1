글꼴 속성

속성 값
font-family 일반폰트명 : 굴림체, 궁서체, ‘Times New Roman’, …
폰트패밀리
  • serif : 삐침 있는 명조계열의 글꼴
  • sans-serif : 삐침 없고 굵기가 일정한 고딕계열의 글꼴
  • monospace : 글자 폭과 간격이 일정한 글꼴
  • cursive : 손으로 쓴 것 같은 필기계열의 글꼴
  • fantasy : 화려한 글꼴
font-size(글자크기지정) 절대크기 키워드 : xx-small, x-small, small, medium, large, x-large, xx-large
상대크기 키워드 : smaller, larger
절대단위 : m, mm, in, pt, pc, px
상대단위 : em, ex, %
font-sytle nomal, italic, oblique
font-variant(작은대문자로 변환) nomal, small-caps
font-weight 100... 900
normal==400, bold==700, bolder, lighter
font(글꼴 속성의 일괄지정) [font-style | font-variant | font-weight]
font-size[/ line-height] font-family
@font-face 규칙(사용자가 정의한 글꼴을 사용하기 위한 것) @font-face{font-family:"사용자 지정 글꼴 이름"; src:url("글꼴 경로와 파일명")}
사용가능 글꼴 종류 : .TTF .OTF .EOT .SVG .SVGZ
