배경 관련 속성

속성 설명
background-color 요소의 배경색 지정. <색상>, transparent
background-image 요소의 배경으로 이미지 파일 지정. url('URL'), none
background-repeat 배경이미지의 반복 여부 결정. repeat, repeat-x, repeat-y, space, round
background-attachment 스크롤에 따른 배경이미지의 움직임 여부 지정. scroll, fixed, local
background-position 배경 이미지의 시작 위치 지정. <x키워드 y키워드>, <x% y%>, <xpos ypos>
background-clip 배경속성이 적용되는 영역지정. border-box, padding-box, content-box
background-origin 배경이미지가 시작하는 기준 위치 지정. padding-box, border-box, content-box
background-size 배경 이미지의 크기 지정. auto, <폭 높이>, <폭% 높이%>, cover, contain
background 배경관련 속성(background-*)의 일괄지정

background-image 속성

  • 요소의 배경으로 사용할 이미지 파일 지정
  • 값 → none, url(URL)

background-repeat 속성

  • 배경 이미지의 반복 여부 지정
  • 값 → repeat, repeat-x, repeat-y, no-repeat, space, round
    space : 배경을 채우는 영역의 경계에서 이미지가 잘려보이지 않도록 반복 이미지간의 공백조정
    round : 배경을 채우는 영역의 경계에서 이미지가 잘려보이지 않도록 이미지의 크기를 재설정

background-attachment 속성

  • 배경 이미지를 콘텐츠 영역과 함께 스크롤되도록 할 것인지의 여부 지정
  • 값 → scroll, fixed, local
    scroll : 배경이미지가 콘텐츠 영역과 함께 스크롤 되도록 지정
    fixed : 배경 이미지는 뷰포트에 고정되어 있고 콘텐츠 영역만 스크롤 되도록 지정
    local : 요소 내부에서 스크롤이 발생할 경우, 요소 내부의 콘텐츠 영역과 함께 스크롤 되도록 지정

background-position 속성

  • 배경 이미지의 시작 위치 지정
  • 기본위치 → 요소의 왼쪽 상단 모서리

background-origin 속성

  • 배경 이미지가 시작하는 기준 위치 지정
  • 값 → padding-box, border-box, content-box
  • background-origin 속성은 배경 이미지가 시작하는 기준 위치를 지정하며,
    background-attachment 속성이 fixed로 지정되면동작하지 않는다.

background-clip 속성

  • 배경 속성(색상, 이미지)이 적용되는 영역 지정
  • 값 → border-box, padding-box, content-box
  • background-clip 속성은 배경 속성이 적용되는 영역을 지정한다.

background-size 속성

  • 배경 이미지의 크기 지정
  • 값 → auto, <폭_길이 높이_길이>, <폭% 높이%>, cover, contain
    <폭_길이 높이_길이>, <폭% 높이%> : 하나의 값만 사용되면 높이는 auto로 처리
    cover : 이미지의 폭과 높이 중 길이가 짧은 것을 기준으로 영역 채움. 원래 크기 비율 무시. contain : 이미지의 폭과 높이 중 길이가 긴것을 기준으로 영역을 채움. 원래 크기 비율 유지.

background 속성

  • background-* 속성을 일괄적으로 지정
  • background: -color -image -position / -size -repeat -origin -clip -attachment
  • 각 속성은 생략가능
  • background-size 속성의 값은 반드시 '/' 를 사용해서 표시
  • 배경 이미지를 여러 개 사용하면서 배경색을 지정하는 경우
    background-color 속성의 값을 리스트의 맨 뒤에 위치시킴

  • 한 번에 여러 개의 배경 이미지를 일괄적으로 지정하는 경우콤마로 구분

속성 설명

disc, circle, square, decimal, lower-alpha, upper-alpha, lower-roman, upper-roman
list-style-position 항목 마커의 위치 지정
inside, outside
list-style-image 이미지를 항목 마커로 지정

list-style 목록속성의 일괄지정(생략된 속성값은 기본값 사용)
list-style:list-style-type값, list-style-position값, list-style-image값;
list-style-type 속성 -> 순서 없는 목록 표시
  • 목록 표시 마크 없음: none
  • 디스크: disc
  • 원:circle
  • 사각형: square

list-style-type 속성 -> 순서 있는 목록 표시
  1. 10진수: decimal
  2. 소문자 로마숫자: lower-roman
  3. 대문자 로마숫자: upper-roman
  4. 소문자 알파벳: lower-alpha
  5. 대문자 알파벳: upper-alpha

테이블 관련 속성

속성 설명
table-layout 셀안의 내용 크기에 따른 셀 크기의 변화여부.
auto → 열의 폭이 셀의 내용의 크기에 따라 결정되므로 테이블 크기가 자동으로 조정됨
fixed → 셀의 내용의 크기와 상관없이 테이블 또는 셀의 폭에 의해서만 결정됨
border-collapse 테이블 테두리와 셀 테두리를 하나의 테두리로 합칠 지의 여부.
separate → 두 테두리를 분리해서 표시
collapse → 두 테두리를 하나로 합쳐서 표시.(border-spacing과 empty-cells 속성이 무시됨)
border-spacing 인접한 셀 테두리 사이의 여백 거리 지정. <수평거리> <수직거리> (0)
border-side 테이블 캡션의 위치 지정.
top → 테이블 상단에 표시
bottom → 테이블 하단에 표시
empty-cells 빈 셀의 테두리와 배경의 표시 여부.
show, hide
