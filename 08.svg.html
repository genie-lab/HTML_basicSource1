SVG
XML 기반의 2차원 벡터 그래픽을 위한 플랫폼
팀 버너스-리 → “모든 크기의 장치에서 최적으로 그림을 그릴 수 있는 그래픽”
Scalable Vector Graphic
웹용 확장 가능한 벡터 그래픽 SVG→“벡터 방식” / 캔버스→“비트맵 방식”

도형 및 텍스트 그리기
사각형 그리기 -> rect 요소 rect x="시작점x" y="시작점y" width="폭" height="높이"

모서리가 둥근 사각형
사각형 그리기 -> rect 요소 rect x="시작점x" y="시작점y" width="폭" height="높이"

원그리기
circle 요소 circle cx="중심 x좌표" cy="중심 y좌표" r="반지름"
cx, cy 생략 → 원의 중심 (0,0)

타원그리기
ellipse 요소 ellipse cx="중심x" cy="중심y" rx="x축반지름" ry="y축반지름"

선그리기
line 요소 line x1="x1" y1="y1" x2="x2" y2="y2"

polyline 요소 polyline points="직선을 구성하는 (x,y) 집합"

다각형 그리기
polygon 요소 polygon points="다각형의 각 꼭지점 (x,y) 집합"

경로 그리기
path 요소 path d="경로 데이터"
-> 경로 데이터에서 사용되는 명령
• Mx,y→ 절대 좌표 (x, y)를 시작점으로 지정
• Lx,y→ 절대 좌표 (x, y)까지 선을 그림
• Z → 마지막 좌표와 처음 좌표를 연결하여 폐쇄 도형을 표시
• 대문자 명령는 절대 좌표, 소문자 명령은 상대 좌표를 의미


2차 베이지 곡선 Q 또는 q 명령

텍스트 그리기
text 요소 text x="시작위치x" y="시작위치y">텍스트


이미지 삽입하기
image 요소 image x="시작위치x" y="시작위치y" height ="높이" weight="폭" xlink:href="이미지경로"

SVG 필터_그림자 효과 기본 연산 → feOffset 요소, feBlend 요소

선형 그라데이션 linearGradient 요소
