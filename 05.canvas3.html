도형변환
캔버스의 좌표계의 변환(이동, 회전, 확대/축소)을 통해 캔버스에 그려지는 도형의 변환을 지정
이동 변환 translate(x, y)

크기 변환1 scale(x, y)

크기 변환2 scale(x, y) 인자가 음수인 경우

회전 변환 rotate(각도)

사용자 정의 변환 transform(), setTransform()

이미지 삽입
drawImage() 메서드
drawImage(이미지변수, dx, dy) 원본 이미지의원래 크기대로 캔버스의 지정된 위치 (dx, dy)에 삽입
drawImage(이미지변수, dx, dy, dw, dh) 원본 이미지를 캔버스의 지정된 위치 (dx, dy)에서dw*dh 크기로 삽입
drawImage(이미지변수, sx, sy, sw, sh, dx, dy, dw, dh) 원본 이미지의 (sx, sy) 위치에서sw*sh 크기로 잘라내어
캔버스의 지정된 위치 (dx, dy)에서dw*dh 크기로 삽입

이미지 데이터조작
이미지의 각 픽셀에 직접 접근해서 다양한 조작이 가능
createImageData(폭, 높이), createImageData(이미지데이터) 모든 픽셀이 투명한 검은색으로 초기화된 빈 ImageData 객체 생성/반환
getImageData(sx, sy, sw, sh) 캔버스 영역에서 지정한 사각형 영역의 픽셀 데이터를 복사한 ImageData 객체를 반환
putImageData(이미지데이터, dx, dy) 지정한 ImageData 객체의 데이터를 캔버스의 지정한 위치에 그림
