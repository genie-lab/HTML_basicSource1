시멘틱 블록요소

main 요소

문서 body 영역의 주요 콘텐츠 블록을 지정할 때 사용
→ 문서 전체에서 반복적으로 사용되는 콘텐츠는 포함 불가
→ 사이드바, 내비게이션 링크, 저작권 정보, 사이트 로고, 검색 폼 등
→ 문서에서 유일한 부분이므로 body 요소 내에서 오직 한 번만 사용
→ 문서의 전체적인 아웃라인에는 아무런 영향을 주지 않음
→ article, aside, footer, header, nav 요소의 하위 요소로 사용 불가

figure 요소

본문에 삽입된 독립적인 콘텐츠를 하나로 묶어 블록을 형성하고, 캡션을 포함시켜 서로의 관계를 명확히 구조화시킬 때 사용 - 요소의 위치는 주요 콘텐츠와 독립적
- figure 요소에 대한 캡션을 표시하는 하위 요소 (그림, 다이어그램, 사진, 소스코드, 동영상등)
- 삭제하더라도 문서의 흐름에는 아무런 영향을 주지 않음
- figcaption 요소
- 위치 → figure 요소 바로 다음 또는 맨 마지막

detail 요소

사용자가 핸들을 클릭함으로써 추가적인 세부정보를 보여 주거나 숨기는 형태의 상호작용을 지원하는 요소 - summary 요소
- 세부정보에 대한 캡션을 지정하기 위한 details 요소의 하위 요소
- 세부정보는 어떤 종류의 콘텐츠도 가능
- open 속성 → 세부정보가 사용자에게 보여지도록 지정
- details 요소의 첫 번째 자식 요소로 사용해야 함
- 생략되면 “세부정보”라는 기본 제목으로 표시
